# 📊 PRODEX - Inteligencia de Datos para PyMEs

¡Bienvenido al repositorio oficial de **PRODEX**! Este proyecto nace como una solución tecnológica diseñada para profesionalizar la gestión de ventas en emprendimientos y pequeñas empresas que aún operan de manera informal.

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma integral que permita recolectar, organizar y analizar datos de ventas para facilitar la toma de decisiones estratégicas basadas en información real y no solo en la intuición.

## 🚀 Funcionalidades Principales

* **Dashboard Estratégico:** Visualización interactiva de indicadores clave (KPIs) como rentabilidad y volumen de ventas.
* **Gestión de Inventario Inteligente:** Registro estructurado de productos y limpieza automatizada de datos para eliminar inconsistencias.
* **Análisis Predictivo (IA):** Modelos de predicción para gestión de inventario y recomendaciones de ventas.
* **Segmentación de Consumidores:** Identificación de patrones de compra por demografía (edad, género, ubicación).

## 🧠 Tecnologías Utilizadas (Metodología SESA)

1. **ChatGPT:** Ampliación de capacidades para la interpretación de datos en lenguaje natural.
2. **TensorFlow:** Solución técnica para modelos de predicción de demanda de stock.
3. **Tableau AI / Chart.js:** Enriquecimiento visual mediante gráficos interactivos.
4. **GitHub Copilot:** Sustitución de tareas repetitivas en la optimización de código.
5. **Tailwind CSS:** Diseño moderno y profesional (UX/UI) adaptado para móviles.

## 📈 Impacto Social

PRODEX permite la **sincronización local (Modo Offline)** para que los pequeños comerciantes en zonas con baja conectividad puedan seguir gestionando sus negocios sin interrupciones.
